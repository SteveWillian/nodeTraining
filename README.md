
# RESTful API

Short repository to describe how to works all kinds of request such as get, post, put and delete

## Run Locally

Clone the project

```bash
  git clone https://github.com/SteveWillian/nodeTraining.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
